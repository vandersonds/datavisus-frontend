O arquivo com seus dados deverá ser salvo na extensão .CSV e ter a mesma formatação da tabela que deu origem ao modelo. Para visualizá-la, acione o menu [Editar | Dados].

Editar dados

Modelo table

Geralmente, a tabela de exemplo tem a quantidade mínima de colunas necessárias para a geração do gráfico. Para importar um novo arquivo utilize o menu [Arquivo | Importar .CSV].

Importar *.CSV

Browser

Após carregar o arquivo com seus dados, será necessário fazer a associação das colunas da tabela com os componentes do gráfico através do menu [Editar | Séries].

Os botões alaranjados representam as séries disponíveis. Para associar uma série a uma coluna de sua tabela, arraste-a sobre o espaço acima do nome da coluna.

Os botões azuis acima dos títulos das colunas contém o nome da série que está associada a esta coluna. Para remover a associação, clique sobre eles.

Editar Séries

Associe as colunas relevantes do arquivo CSV com as séries de dados do documento: arraste o nome da série (botões verdes ou laranja) sobre a coluna que deseja associar. para remover uma associação, clique no botão azul que está sobre a coluna. um botão laranja significa que você precisa associar esta série com pelo menos uma coluna.
